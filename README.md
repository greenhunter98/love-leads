# Love Leads Organization Website

A modern, responsive website for Love Leads Organization, a faith-based non-profit operating in Malawi.

## About Love Leads

Love Leads Organization is a non-governmental and non-profit faith-based entity dedicated to transforming lives in Malawi through:

- **Children**: Early childhood development, school feeding, back-to-school support
- **Youth**: Sports, talent development, HIV/AIDS awareness
- **Women**: Empowering single parents through entrepreneurship
- **Disaster Response**: Emergency relief and community resilience
- **Spiritual**: Evangelism and "The Action Way"
- **Community**: Income generation, counseling, education

## Tech Stack

- [Next.js 14](https://nextjs.org/) - React framework
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Lucide Icons](https://lucide.dev/) - Icon library
- [Heroicons](https://heroicons.com/) - Additional icons
